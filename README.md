Vehicle Rental System (TypeScript OOP)

A simple TypeScript project demonstrating OOP concepts like inheritance, polymorphism, and class interactions.
Vehicles (Car, Bike) extend a base Vehicle class. Customers can rent any vehicle, and the rental service prints dynamic rental details.

Features

Base Vehicle class with specialized Car and Bike

Customer class extending Person

Rental service that handles any vehicle type

Shows polymorphism via getinfo() overrides
